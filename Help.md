# 🟢 GooglyMoogly Discord Bot 사용 가이드

👉 **[구글리의 깃허브에서 보기](https://github.com/joocfresh/joocBot/blob/joocfresh-patch-1/Help.md)**  
봇이 마음에 드셨다면 ⭐ Star 한 번 부탁드립니다.

## 1️⃣ GooglyMoogly 봇이란?

GooglyMoogly 봇은 **Albion Online Killboard 정보를 Discord에서 자동으로 조회·수신**할 수 있는 봇입니다.

- 킬 / 데스 로그 조회
- 특정 플레이어 킬보드 구독
- 채널 단위 킬보드 관리
- Albion 서버 선택 지원


## 2️⃣ 기본 사용 방법 (가장 중요)

GooglyMoogly 봇은 **반드시 멘션(@)과 함께 명령어를 입력해야 작동**합니다.

### 기본 문법
```
@googlymoogly봇 /{명령어} {파라미터}
예시) @googlymoogly봇 /k cp8282
```
❌ 멘션 없이 명령어만 입력하면 작동하지 않습니다.  
⭕ 반드시 `@googlymoogly봇`을 포함하세요.

---

## 3️⃣ 명령어(Command)란?

명령어는 봇에게 **무엇을 할지 지시하는 키워드**입니다.  
대부분의 명령어는 **동의어(축약어)** 를 함께 지원합니다.

---

## 4️⃣ 전체 명령어 목록

### 🔧 기본 / 안내
| 명령어 | 설명 | 동의어 |
|------|------|------|
| `/테스트` | 봇이 살아있는지 테스트 (따라 말함) | `/e`, `/echo`, `/에코` |
| `/도움말` | 도움말 표시 | `하이구글리`, `/h`, `/help` |
| `/패치노트` | 패치 노트 표시 | `/p`, `/patch`, `/패치` |

---

### ⚔️ 킬보드 / 로그
| 명령어 | 설명 | 동의어 |
|------|------|------|
| `/킬로그` | 가장 최근 킬/데스 로그 조회 | `/k`, `/killlog`, `/킬` |
| `/검색` | 플레이어 검색 (ID 확인용) | `/search` |

---

### 📌 구독 / 관리 (승인 필요)
| 명령어 | 설명 | 동의어 |
|------|------|------|
| `/구독` | 현재 채널에서 킬보드 수신 시작 | `/subscribe` |
| `/구독중지` | 킬보드 수신 중지 | `/unsubscribe`, `/구독취소` |
| `/디스밴드` | 채널의 킬보드 인원 전체 삭제 | `/clear`, `/disband`, `/청소` |
| `/제명 {이름}` | 특정 플레이어 1명 삭제 | `/제거`, `/delete` |
| `/등록 {이름}` | 현재 등록된 플레이어 목록 조회 | `/r`, `/register` |

---

### ⚙️ 환경 / 정보
| 명령어 | 설명 | 동의어 |
|------|------|------|
| `/서버` | Albion 서버 설정 (기본: 동부 서버) | `/server` |
| `/채널` | 현재 채널 정보 출력 | `/channel` |

---

## 5️⃣ 파라미터(Parameter)란?

파라미터는 **명령어가 적용될 대상**입니다.  
주로 **플레이어 이름**을 사용합니다.

### 파라미터 사용 예시
```
@googlymoogly봇 /등록 Googlymoogly5404
@googlymoogly봇 /제명 Googlymoogly5404
@googlymoogly봇 /킬로그 Googlymoogly5404
@googlymoogly봇 /검색 Googlymoogly5404
@googlymoogly봇 /에코 안녕하세요
```

⚠️ Albion Online은 **닉네임이 아닌 고유 ID 기반**이므로  
정확한 등록을 위해 `/검색` 명령어 사용을 권장합니다.

---

## 6️⃣ 자주 발생하는 실수

- ❌ 멘션 없이 명령어만 입력함
- ❌ `/제명`을 파라미터 없이 사용
- ❌ 서버 설정 없이 다른 서버 로그를 기대함
